## 📖 项目简介

本项目整理了 2022 届 MBA 备考过程中的择校信息和复习资料，旨在帮助考生高效备考、科学择校。

**备考目标**：
- 总分：300 分 → 目标 200 分
- 英语：≥ 45 分

## 📚 备考资料

### 考试科目

1. **管理类联考综合能力** (200 分)
   - 数学：75 分（初高中数学知识）
   - 逻辑：60 分（推理、论证）
   - 写作：65 分（论证有效性分析 + 论说文）

2. **英语二** (100 分)
   - 完形填空：10 分
   - 阅读理解：50 分
   - 翻译：15 分
   - 写作：25 分

### 推荐资料

- **数学**：《数学分册》、《高分指南》
- **逻辑**：《逻辑精点》、《逻辑 1000 题》
- **写作**：《写作分册》、《历年真题范文》
- **英语**：《考研英语二真题》、《单词红宝书》

## 📅 备考时间线

```
3-6 月  │ 基础阶段：词汇 + 数学基础
7-9 月  │ 强化阶段：刷题 + 专项突破
10-11 月│ 冲刺阶段：真题 + 模拟
12 月   │ 考前调整：查漏补缺
```

## 💡 备考建议

### 数学
- 重视基础，不要盲目追求难题
- 建立错题本，定期复习
- 掌握快速解题技巧

### 逻辑
- 理解题干，识别论证结构
- 多做题，培养题感
- 总结常见题型和解题套路

### 写作
- 积累素材，关注时事
- 练习框架，形成自己的模板
- 多写多改，找人点评

### 英语
- 每天背单词，保持语感
- 精读真题，分析长难句
- 作文多背范文，积累好句
